# Glossary

| Term | Meaning |
| ---- | ------- |
| **Host repo** | Product repository consuming nerd-style |
| **Pocket-protector** | Meticulous engineering culture without cruelty |
| **Thin handler** | Entry that only validates, authorizes, delegates, responds |
| **Living SPEC** | Spec updated in the same change set as behavior |
| **Idle cost** | What you pay when nobody uses the system |
| **Pre-push gate** | Format + lint before every push |
| **Opaque token** | Random secret id (guest/signing), not a JWT |
| **Soft delete** | Mark deleted/inactive; retain row for recovery/audit |
| **UI tokens** | CSS variables for brand/surface/status (`configs/ui/tokens.css`) |
| **Portal shell** | Authenticated topbar + sidebar + main layout pattern |
